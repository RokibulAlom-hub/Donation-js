What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, where each node corresponds to a part of the document (such as elements, attributes, and text).

How do you select an element from the DOM?

In JavaScript, there are several ways to select elements from the DOM. Here are the most common methods: getelemntbyid(),getElementsByClassName(),getElementsByTagName(),querySelector(),querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

When an event occurs on an element (such as a click event on a button), the event first happens on the element itself and then bubbles up through its ancestors (parent, grandparent, etc.) until it reaches the document object. This is known as event bubbling.

How do you manipulate an element's attributes and styles using the DOM?

In JavaScript, you can manipulate an element's attributes and styles using the DOM in various ways. Here's how you can do it: getAttribute(),setAttribute(),removeAttribute()Some common attributes, like id, class, value, src, and href, can be accessed or modified directly through properties.
